## INSTALLATION

install nodeJS

$ npm install  
$ cd client  
$ npm install  
$ cd ../server  
$ npm install  
$ cd ..

# RUN

\$ npm start

or separately  
$ node start-client  
$ node start-server

# INFOS

client created with create-react-app  
server created with Express  
assets from lichess.org

client runs on http://localhost:3000/  
server runs on http://localhost:3001/
